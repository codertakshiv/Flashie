# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Flashie, please report it privately by opening a security advisory at:

https://github.com/flashie/flashie/security/advisories

Please do not disclose the vulnerability publicly until it has been addressed.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |
